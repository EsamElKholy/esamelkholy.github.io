TOOL 01

UI Studio

WYSIWYG uGUI authoring + screen routing · adopted by 3 projects

A standalone Unity package that turns uGUI screen-building into a visual workflow. The editor side pairs a Flow Graph — a node graph of every screen and the navigation between them, with per-screen breakpoints and a Play-Mode navigation breadcrumb — with a WYSIWYG Screen Editor (anchor presets, smart guides, drag/resize, multi-resolution preview) and familiar Hierarchy/Properties/Palette panels. The runtime side is a ScreenManager that owns the transition pipeline (fade, timed/manual wait, custom) with ScriptableObject-based screen graphs and transitions. Teams stand up and rewire a whole game flow visually instead of hand-coding scene changes.

TOOL 02

VN Engine & Story Navigator

Reusable VN engine + Yarn story-graph tool · ~50% less debugging/testing time, reused across 2 projects

A reusable, installable Unity visual novel engine (localization-aware, with a sample VN included). With a standout tool which is the Story Navigator: it parses .yarn files into a directed dependency graph of story nodes and renders them on an interactive canvas with filtering, auto-layout, sticky-note annotation, pathfinding between nodes, a play-mode debug runner, and save-file integration — turning opaque branching dialogue into a graph the whole team can read, trace, and debug. Built on a clean three-tier architecture (framework → VN domain → game extension).

Unity PackageYarn SpinnerNode GraphLocalizationSave Integration
TOOL 03

Node Canvas — Reusable Editor Framework

One framework, two shipping tools

A generic, domain-free node-graph framework for Unity editor tools, built on UI Toolkit with a generics-based core (NodeGraphCanvas<TNode,TEdge>): pan/zoom viewport with level-of-detail culling, bezier/polyline edges with reroute points, collapsible group bands, an undo/redo command stack, box and multi-select, a persistence hook for custom layouts, and a metadata layer for category badges and isolation dimming. It has zero references to any game code. This is the shared foundation under both UI Studio's Flow Graph and the Story Navigator, and reusable for quest editors, state machines, or skill trees.

TOOL 04

Jenkins Build Tool & Console Pipeline

Unity↔Jenkins CI/CD + PS5/Switch support · adopted by 5 projects, ~70% faster build & test

A Unity editor tool that bridges Unity and Jenkins: trigger and monitor builds without leaving the editor, a template-based Jenkinsfile generator, and a Quick-Build mode that auto-creates temporary commits and timestamped CI branches — then cleans them up — so you can build uncommitted work in seconds. Live build history, Slack notifications, and multi-provider Git integration. I extended it with full PS5 and Switch console support: a unified cross-platform save architecture with AES encryption and double-buffering, DualSense input, and Switch nn.fs filesystem handling.
